## ¡Hola, soy Jair Macedo! 👋
Soy Data Analyst especializado en inteligencia de decisiones, modelado predictivo y análisis de retail. Mi enfoque combina la curiosidad técnica con una visión clara orientada a resultados de negocio.

Actualmente, ayudo a empresas a tomar decisiones estratégicas basadas en datos mediante el uso de modelos estadísticos, visualización de información y limpieza de bases de datos.

🚀 Sobre mí
Enfoque: Transformo datos complejos en insights accionables para la toma de decisiones empresariales.

Experiencia: He colaborado en proyectos clave como el análisis de riesgos en el sector automotriz y el desarrollo de estrategias para consumo masivo (Reckitt México).

Pasión: Me apasiona el análisis de rendimiento y las tendencias, actualmente enfocado en modelado predictivo para el ecosistema deportivo (Mundial 2026).

🛠️ Tech Stack (Mi caja de herramientas)
Análisis y Modelado: Python (Pandas, Scikit-learn), SQL, Excel avanzado.

Visualización: Looker Data Studio, Power BI.

Entorno de trabajo: Jupyter Notebooks, VS Code, Git/GitHub.

📊 Proyectos Destacados:
 Análisis de rendimiento, proyecciones y tendencias de jugadores para el Mundial 2026.

 Modelado de riesgos y análisis para optimización de procesos.
